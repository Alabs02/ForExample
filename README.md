# ForExample
ForEx Trading WebApp that serves as an example that demonstrates Mithril+Hasura.

## Pre-requisites

- Docker
- Docker-compose

## Installation

- ### BackEnd
  In the root directory:
  - `docker-compose build`
  - `docker-compose up -d`

- ### FrontEnd
  In the `mithrilClient` directory:
  - `npm install`
  - `npm start`
